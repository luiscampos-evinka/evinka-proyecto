# EVINKA Suite

App principal Flutter de EVINKA.

## Complementos
- `README_INTEGRACION.md` → flujo integrado con cotizador
- `INSTRUCCIONES_PARA_COMPILAR.md` → compilación

## Ejecutar
```bash
/opt/flutter/bin/flutter pub get
/opt/flutter/bin/flutter run
```

## Build release
```bash
JAVA_HOME=/usr/lib/jvm/java-17-openjdk-amd64 /opt/flutter/bin/flutter build apk --release
```
