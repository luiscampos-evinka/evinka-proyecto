# Start here

Si vas a tocar código en EVINKA:

1. Lee `docs/ARCHITECTURE.md`
2. Lee `docs/DEPLOYMENT.md`
3. Mira el README de la app que vas a tocar
4. Mantén cada frente separado
5. No metas data viva ni builds al repo

## Orden recomendado
1. EVINKA Suite
2. Cotizador web
3. MapCo / Overview
4. Apps satélite
